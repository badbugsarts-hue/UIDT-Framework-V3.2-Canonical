‚ö° Quick Start (Reproducibility)

3.1 Prerequisites

Requires Python 3.8+ and standard scientific libraries. For GPU acceleration, CuPy is required.

# Recommended environment setup
conda create -n uidt_env python=3.10
conda activate uidt_env

3.2 Installation

Install necessary dependencies from the provided file:

pip install -r requirements.txt
# If GPU acceleration is desired:
# pip install cupy-cudaXX

3.3 Execution

Run the primary diagnostic script to perform the $\mathbf{\kappa}$-scan, continuum limit extrapolation, and $\mathbf{0^{++}}$ Mass Gap calculation:

python src/UIDTv3.2_Hmc-Simulaton-Diagnostik.py

This script will generate all core plots and results in the RESULTS/ folder.

3.4 Verification

Run the test suite for numerical stability and correctness of the highly optimized routines (e.g., $\mathbf{SU(3)}$ matrix exponential via Cayley-Hamilton):

python -m pytest tests/
