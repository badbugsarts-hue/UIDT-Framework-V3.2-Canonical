    <h2>1. Overview</h2>
    <p>This repository contains the complete dataset from the final <strong>Monte Carlo (MC) validation run</strong> for the Unified Information-Density Theory (UIDT) $\Omega$ framework. This data serves as the non-perturbative statistical validation of the canonical parameters ($\mathbf{\Delta, \gamma, \kappa, m_S, \lambda_S}$) derived from the analytical 3-equation system.</p>
    <p>The dataset includes the raw MC chain samples, high-precision mean values, parameter correlation matrices, and the final summary tables and plots used in the UIDT v3.3 publication.</p>
    
    <h3>üåü Key Data Artifacts</h3>
    <ul>
        <li><strong>Raw Data:</strong> 100,000 high-thinning samples from the primary MC simulation (<code>UIDT_MonteCarlo_samples_100k.csv</code>).</li>
        <li><strong>High-Precision Means:</strong> Final derived parameter values with high precision (<code>UIDT_HighPrecision_mean_values.csv</code>).</li>
        <li><strong>Correlation Analysis:</strong> Full correlation matrix showing dependencies between parameters (<code>UIDT_MonteCarlo_correlation_matrix.csv</code>).</li>
        <li><strong>Plots:</strong> Diagnostic plots including posterior distributions (corner plot) and parameter histograms.</li>
    </ul>

    <h2>3. How to Use (Reproducibility)</h2>
    <p>To reproduce the analysis and plots from this dataset:</p>

    <ol>
        <li><strong>Load the Samples:</strong>
<pre><code>import pandas as pd
# Load the primary data source
samples = pd.read_csv("UIDT_MonteCarlo_samples_100k.csv")
</code></pre>
        </li>
        <li><strong>Verify High-Precision Means:</strong>
<pre><code># Calculate the mean for a parameter, e.g., 'gamma'
mean_gamma = samples['gamma'].mean()
std_gamma = samples['gamma'].std()
print(f"Calculated gamma: {mean_gamma:.4f} +/- {std_gamma:.4f}")

# Compare against the high-precision file
hp_means = pd.read_csv("UIDT_HighPrecision_mean_values.csv", index_col=0)
print(f"Stored mean: {hp_means.loc['gamma', 'mean']:.4f}")
</code></pre>
        </li>
        <li><strong>Reproduce Plots:</strong>
            <p>Use libraries like <code>matplotlib</code>, <code>seaborn</code>, or <code>corner</code> to regenerate the plots.</p>
            <p><strong>Example (Corner Plot):</strong></p>
<pre><code>import corner
# Ensure 'corner' is installed: pip install corner
fig = corner.corner(samples, labels=samples.columns, 
                    quantiles=[0.16, 0.5, 0.84], 
                    show_titles=True, title_fmt=".4f")
fig.savefig("REPRODUCED_corner_plot.png")
</code></pre>
        </li>
    </ol>
